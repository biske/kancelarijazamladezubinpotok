== Подешавање

=== Клонирање

=== Подешавање базе

Преименовати .env.example у .env и подесити одговарајуће параметре.

Покренути команду:

bin/rake db:create db:migrate

# Покретање апликације

bin/rails s

Апликација је доступна на адреси http://localhost:3000
